CRUD Application (Student Management System / Notes App)

I’ve successfully built a full CRUD (Create, Read, Update, Delete) application as part of my Web Development learning journey.

💡 Project Overview:
This project is a simple Student Management System where users can:

➕ Add student data
📄 View all records
✏️ Edit existing data
❌ Delete records

🛠️ Tech Stack Used:

Frontend: HTML, CSS, JavaScript
Backend: Node.js with Express
Database: MongoDB (Atlas)
API Testing: REST API

⚙️ Core Features Implemented:

REST API creation:
GET /api/items (fetch data)
POST /api/items (add data)
PUT /api/items/:id (update data)
DELETE /api/items/:id (remove data)
Frontend integration using Fetch API
Dynamic table rendering for real-time updates
Full backend + database connection
